🎯 Purpose of this Repository

This repository is created as **my personal lab** and, at the same time, a **portfolio** to demonstrate my skills in network traffic analysis. I have set the following goals for myself:

 1. Educational Goals
- Deepen understanding of network protocols (HTTP, DNS, ARP, TCP, ICMP, DHCP).
- Learn to quickly detect anomalies and attack indicators in large traffic volumes.
- Master advanced filtering and statistical analysis techniques in Wireshark.
- Study automation of analysis using `tshark` and Python.

 2. Practical Skills
- Practice incident response scenarios (as in a SOC).
- Learn to identify MITM attacks (ARP‑spoofing, DHCP attacks).
- Master threat hunting using malicious traffic datasets.
- Explore data exfiltration methods and C2 channels.

 3. Demonstrating Competence to Employers
- Show that I can not only use a tool but also **think analytically**.
- Document every step: from hypothesis to conclusions and recommendations.
- Demonstrate a systematic approach to solving cybersecurity tasks.
- Create a clear portfolio that speaks for me during interviews.

 4. Contribution to Professional Growth
- Build a knowledge base that I can return to for quick reference.
- Share findings with the community and receive feedback.
- Continuously update the repository with new cases to track my progress.
